burrow

A platform-agnostic, directory-scoped secrets manager. Store secrets outside your repos, inherit them through directory ancestry.

curl -fsSL https://burrow.dev/install.sh | sh

Or download from GitHub Releases

~/projects/myapp

Why burrow?

🔒

Secrets Stay Safe

Secrets are stored in your user profile, never in your repo. No more .env files to accidentally commit.

📂

Directory Scoped

Set secrets for any directory. Child directories automatically inherit from parents.

🌐

Cross Platform

Works on Linux, macOS, and Windows. Single binary, no dependencies.

🛠

Unix Philosophy

Exports environment variables for shell composition. Works with any toolchain.

Usage

Set a secret

Store a secret scoped to the current directory (or specify a path).

burrow set API_KEY=sk-abc123
burrow set DATABASE_URL=postgres://... --path ~/projects

Get a secret

Retrieve a secret, resolved through directory ancestry.

burrow get API_KEY --show
burrow get API_KEY --format json

List all secrets

See all resolved secrets for the current directory.

burrow list
burrow list --format json

Export to your shell

Load secrets into your environment with a single command.

eval "$(burrow export)"
eval "$(burrow export --format shell)" && npm start

Block inheritance

Prevent a parent secret from being inherited in a child directory.

burrow unset API_KEY --path ~/projects/myapp/tests

Commands

Command Description
set KEY=VALUE Set a secret at the current or specified path
get KEY Get a secret resolved from directory ancestry
list List all resolved secrets for current directory
unset KEY Block a secret with a tombstone (prevents inheritance)
export Export secrets in shell, dotenv, or JSON format
